# 4 寻找两个有序数组的中位数

## [LeetCode](https://leetcode-cn.com/problems/median-of-two-sorted-arrays/)

## 题目

给定两个大小为 m 和 n 的有序数组 nums1 和 nums2。

请你找出这两个有序数组的中位数，并且要求算法的时间复杂度为 O(log(m + n))。

你可以假设 nums1 和 nums2 不会同时为空。

## 示例

示例 1:

```text
nums1 = [1, 3]
nums2 = [2]

则中位数是 2.0
```

示例 2:

```text
nums1 = [1, 2]
nums2 = [3, 4]

则中位数是 (2 + 3)/2 = 2.5
```

## 解答

```java
class Solution {
    public double findMedianSortedArrays(int[] nums1, int[] nums2) {
        int index = (nums1.length + nums2.length) / 2;
        // 两个数组总和之长为奇数
        if ((nums1.length + nums2.length) % 2 != 0) {
            int n1 = 0;
            int n2 = 0;
            int cur = 0;
            while (true) {
                if ((n1 + n2) > index) {
                    return cur;
                }
                // 依次寻找两数组中较小的那个元素
                if (n2 >= nums2.length) {
                    cur = nums1[n1];
                    n1++;
                } else if (n1 >= nums1.length) {
                    cur = nums2[n2];
                    n2++;
                } else if (nums1[n1] < nums2[n2]) {
                    cur = nums1[n1];
                    n1++;
                } else {
                    cur = nums2[n2];
                    n2++;
                } 
            }
        } else {
            int n1 = 0;
            int n2 = 0;
            int cur = 0;
            int pre = 0;
            while (true) {
                if ((n1 + n2) > index) {
                    return (cur + pre) / 2.0;
                }
                // 依次寻找两数组中较小的那个元素
                if (n2 >= nums2.length) {
                    pre = cur;
                    cur = nums1[n1];
                    n1++;
                } else if (n1 >= nums1.length) {
                    pre = cur;
                    cur = nums2[n2];
                    n2++;
                } else if (nums1[n1] < nums2[n2]) {
                    pre = cur;
                    cur = nums1[n1];
                    n1++;
                } else {
                    pre = cur;
                    cur = nums2[n2];
                    n2++;
                } 
            }
        }
    }
}
```
