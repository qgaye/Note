# 11 盛最多水的容器

```java
class Solution {
    public int maxArea(int[] height) {
        int i = 0;
        int j = height.length - 1;
        int res = 0;
        while (i < j) {
            if (height[i] < height[j]) {
                res = Math.max(res, height[i] * (j - i));
                i += 1;
            } else {
                res = Math.max(res, height[j] * (j - i));
                j -= 1;
            }
        }
        return res;
    }
}
```
