# 15 三数之和

排序后，确定一个值后用双指针找

注意：
1. 确定第一个值时重复的需要排除
2. 当找到条件后需要继续查找，不能跳出循环

```java
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> res = new ArrayList<>();
        Arrays.sort(nums);
        for (int s = 0; s < nums.length - 2; s++) {
            if (s != 0 && nums[s] == nums[s - 1]) continue;
            int i = s + 1;
            int j = nums.length - 1;
            while (i < j) {
                int sum = nums[s] + nums[i] + nums[j];
                if (sum == 0) {
                    res.add(List.of(nums[s], nums[i], nums[j]));
                    i += 1;
                    while (i < nums.length && nums[i] == nums[i - 1]) i += 1;
                    j -= 1;
                    while (j >= 0 && nums[j] == nums[j + 1]) j -= 1;
                } else if (sum < 0) i += 1;
                else j -= 1;
            }
        }
        return res;
    }
}
```
