# 27 移除元素

```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int count = 0;
        for (int i = 0; i < nums.length - count; i++) {
            if (nums[i] == val) {
                nums[i] = nums[nums.length - count - 1];
                i -= 1;
                count += 1;
            }
        }
        return nums.length - count;
    }
}
```
