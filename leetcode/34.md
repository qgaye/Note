# 34 在排序数组中查找元素的第一个和最后一个位置

二分查找 

```java
class Solution {
    public int[] searchRange(int[] nums, int target) {
        if (nums.length == 0) return new int[] {-1, -1};
        int[] res = new int[] {-1, -1};
        int l = 0;
        int r = nums.length - 1;
        while (l < r) {
            int m = (l + r) / 2;
            if (nums[m] < target) l = m + 1;
            else r = m;
        }
        if (nums[l] != target) return res;
        res[0] = l;
        r = nums.length;
        while (l < r) {
            int m = (l + r) / 2;
            if (nums[m] <= target) l = m + 1;
            else r = m;
        }
        res[1] = l - 1;
        return res;
    }
}
```
