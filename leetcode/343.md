# 343 整数拆分

动态规划

```java
class Solution {
    public int integerBreak(int n) {
        int[] arr = new int[n + 1];
        arr[1] = 1;
        arr[2] = 1;
        for(int i = 3; i <= n; i++) {
            int max = -1;
            for(int j = 2; j < i; j++) {
                int res = Math.max(j * (i - j), j * arr[i - j]);
                if (res > max) max = res;
            }
            arr[i] = max;
        }
        return arr[n];
    }
}
```
