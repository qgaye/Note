# 53 最大子序和

动态规划

```java
class Solution {
    public int maxSubArray(int[] nums) {
        if (nums.length == 0) return 0;
        int max = nums[0];
        for (int i = 1; i < nums.length; i++) {
            if (nums[i - 1] > 0) nums[i] = nums[i] + nums[i - 1];
            max = Math.max(max, nums[i - 1]);
        }
        return max;
    }
}
```
