# 69 x 的平方根

二分查找法

```java
class Solution {
    public int mySqrt(int x) {
        if (x <= 1) return x;
        int l = 1;
        int r = x / 2 + 1;
        while (l <= r) {
            int m = r + (l - r) / 2;
            long mm = (long) m * m;
            if (mm == x) return m;
            else if (mm > x) r = m - 1;
            else l = m + 1;
        }
        return r;
    }
}
```
