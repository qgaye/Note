# 06 从尾到头打印链表

递归或者用Stack

```java
class Solution {
    List<Integer> list = new ArrayList<>();
    public int[] reversePrint(ListNode head) {
        if (head == null) return new int[] {};
        if (head.next != null) {
            reversePrint(head.next);
        }
        list.add(head.val);
        int[] res = new int[list.size()];
        for (int i = 0; i < list.size(); i++) {
            res[i] = list.get(i);
        }
        return res;
    }
}
```
