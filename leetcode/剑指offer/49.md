# 49 丑数

动态规划 丑数 = 2^x * 3^y * 5^z

```java
class Solution {
    public int nthUglyNumber(int n) {
        int[] arr = new int[n + 1];
        arr[1] = 1;
        int p1 = 1;
        int p2 = 1;
        int p3 = 1;
        for (int i = 2; i <= n; i++) {
            arr[i] = Math.min(arr[p1] * 2, Math.min(arr[p2] * 3, arr[p3] * 5));
            if (arr[p1] * 2 == arr[i]) p1 += 1;
            if (arr[p2] * 3 == arr[i]) p2 += 1;
            if (arr[p3] * 5 == arr[i]) p3 += 1;
        }
        return arr[n];
    }
}
```

优先队列来保证大小顺序，Set来保证不重复
