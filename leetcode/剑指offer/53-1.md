# 53-1 在排序数组中查找数字 I

```java
class Solution {
    public int search(int[] nums, int target) {
        // 最后一个target的后一位index - 第一个target的index
        return help(nums, target) - help(nums, target - 1);
    }
    private int help(int[] nums, int target) {
        int l = 0;
        int r = nums.length - 1;
        while (l <= r) {
            int m = l + (r - l) / 2;
            if (nums[m] <= target) l = m + 1;
            else r = m - 1;
        }
        return l;
    }
}
```
