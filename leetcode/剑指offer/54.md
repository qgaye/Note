# 54 二叉搜索树的第k大节点

中序遍历

```java
class Solution {
    public int kthLargest(TreeNode root, int k) {
        List<Integer> list = new ArrayList<>();
        help(root, list);
        return list.get(list.size() - k);
    }
    private void help(TreeNode root, List<Integer> list) {
        if (root == null) return;
        help(root.left, list);
        list.add(root.val);
        help(root.right, list);
    }
}
```
