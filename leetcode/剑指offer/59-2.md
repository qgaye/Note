# 59-2 队列的最大值

用个max维护最大值

```java
class MaxQueue {
    LinkedList<Integer> queue;
    int max;

    public MaxQueue() {
        queue = new LinkedList<>();
    }
    
    public int max_value() {
        return queue.isEmpty() ? -1 : max;
    }
    
    public void push_back(int value) {
        queue.add(value);
        if (value > max) max = value;
    }
    
    public int pop_front() {
        if (queue.isEmpty()) return -1;
        if (max == queue.getFirst()) {
            max = -1;
            for (int i = 1; i < queue.size(); i++) {
                if (queue.get(i) > max) max = queue.get(i);
            }
        }
        return queue.removeFirst();
    }
}
```

滑动窗口，模版，双队列


```java
class MaxQueue {
    LinkedList<Integer> queue;
    LinkedList<Integer> help;

    public MaxQueue() {
        queue = new LinkedList<>();
        help = new LinkedList<>();
    }
    
    public int max_value() {
        return queue.isEmpty() ? -1 : help.getFirst();
    }
    
    public void push_back(int value) {
        queue.addLast(value);
        while (!help.isEmpty() && value > help.getLast()) help.removeLast();
        help.addLast(value);
    }
    
    public int pop_front() {
        if (queue.isEmpty()) return -1;
        if (help.getFirst() == queue.getFirst()) help.removeFirst();
        return queue.removeFirst();
    }
}
```
